# Image Carver

Currently a console-only C++23 application aiming to recreate classic paint-style image editing with an extended toolset.

The project targets all major desktop platforms (Windows, macOS, Linux).

## Current Features

- Create bitmaps using styling objects (lines, rectangles, ovals)
- Export bitmaps as `.bmp` files
- Save drawing settings (`.crv` files)
- Load drawing settings
